Authentication Service is needed to generate tokens and create authentication for each user session

Guard Service is needed to protect sensitive data by checking for login before allowing access to a page
- This is technically optional

HTTPIntercept changes HTTPRequests into more usable data (HTTP Objects)

Guides being followed: 
- No JWT (Local Storage): https://www.tutorialspoint.com/angular8/angular8_authentication_and_authorization.htm#
- Angular + Express + JWT + HttpClient: https://blog.angular-university.io/angular-jwt-authentication/
